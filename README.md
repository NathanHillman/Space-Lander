# Space-Lander
A project from my Numerical Analysis class from Junior year, done in honor of the Artemis II mission.

# Before Playing
Main.py and the files directory must be in the same folder. Make sure you have the necessary packages installed:
- pygame
- matplotlib.pyplot

# How to Play
Simply run main.py. 
Currently a command line interface is used to enter in the desired difficulty & integration method, but after this all rule explanations and gameplay are confined to the pygame window.
